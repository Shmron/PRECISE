Population-Weighted Methodology Example

Environmental exposures are calculated using Carto's H3 hexagonal tessellation to account for heterogeneous population distribution within village boundaries. Each metric is weighted by the population residing in each grid cell, ensuring that densely populated areas have proportionally greater influence on the village-level estimate.

FORMULA: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ NDVI = (NIR - RED) / (NIR + RED) Where: • NIR = Near-Infrared band (Band 5 for Landsat 8/9) • RED = Visible Red band (Band 4 for Landsat 8/9) DATASET PARAMETERS: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ • Collection: Landsat Collection 2 Tier 1 Level 2 • Temporal resolution: 8-day composite • Spatial resolution: 30m • Cloud masking: Applied using QA_PIXEL band GOOGLE EARTH ENGINE CODE: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ var landsat = ee.ImageCollection("LANDSAT/LC08/C02/T1_L2") .filterDate(startDate, endDate) .map(function(img) { var nir = img.select('SR_B5'); var red = img.select('SR_B4'); return img.addBands(nir.subtract(red) .divide(nir.add(red)).rename('NDVI')); }); var ndvi = landsat.select('NDVI').mean().reduceRegion({ reducer: ee.Reducer.mean(), geometry: village.geometry(), scale: 30 }); INTERPRETATION: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ • -1 to 0: Water, bare soil, built-up areas • 0 to 0.2: Sparse vegetation, arid land • 0.2-0.4: Moderate vegetation (shrubs, grassland) • 0.4-0.6: Healthy vegetation (agriculture) • 0.6-1.0: Dense vegetation (forests)

Methodology

Non-fire smoke PM2.5 component is spatially and temporally derived from FMI's global smoke PM2.5 dataset distributed as NC4 (NetCDF4) files using nearest-neighbor interpolation. This represents background particulate matter from sources including traffic, industry, dust, and domestic cooking.

RQI (0 to 1, dimensionless)
Mean ± SD0.288 ± 0.12
Median [Q1, Q3]0.27 [0.20, 0.35]
Range0.08 – 0.67

Dataset

Country-specific road datasets with actual speed attributes:
• Kenya: kenya_roads.shp (fclass, maxspeed)
• Gambia: gambia_roads211021.shp (speed_driv, road_class)
• Mozambique: Mozambique_roads211021.shp (speed_driv, CLASS)

Methodology

Non-population weighted measure of road infrastructure quality within village boundaries. Uses actual driving speed attributes from high-quality road datasets.

RQI = (Weighted Average Speed) / (Reference Speed) = (Σ(road_length × speed)) / (Σroad_length) / 120 CALCULATION STEPS: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 1. Clip road network to village boundary 2. For each road segment within village: - Get length in kilometers - Get speed from speed_driv attribute (or road class) 3. Calculate length-weighted average speed 4. Divide by reference speed (120 km/h) to normalize SPEED DATA SOURCES: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Kenya: maxspeed attribute, fallback to fclass-based speeds Gambia: speed_driv attribute, fallback to road_class Mozambique: speed_driv attribute, fallback to CLASS ROAD TYPE EXCLUSIONS: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Excluded from calculation (not vehicle infrastructure): • Paths, footways, pedestrian ways • Steps, cycleways, bridleways • Trails INTERPRETATION: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ • RQI = 1.0: All roads are motorway-quality (120 km/h) • RQI = 0.5: Average road speed is 60 km/h • RQI = 0.25: Average road speed is 30 km/h • Higher RQI = better road infrastructure quality NOTE: NOT population-weighted (measures infrastructure, not accessibility)

Rationale

Road quality affects travel times and safety when accessing health facilities. Poor quality roads during wet season become flooded and impassable. RQI captures the quality of road infrastructure independent of where people live within the village.

PW_PubTrans_Time = Σ(Popi × TotalTimei) / ΣPopi Where TotalTime = WalkToPickup + WaitTime + InVehicleTime COMPONENT BREAKDOWN: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 1. WALK TO PICKUP POINT (WalkToPickup) • Walking distance from hexagon centroid to nearest public transport route (primary/secondary roads) • Walking speed: 5 km/h • Public transport (matatus, buses) only operates on main roads, not minor/residential roads 2. SIMULATED WAIT TIME (WaitTime) • Average wait time for public transport arrival • Estimated: 15-30 minutes (varies by location) • Rural areas: longer waits due to lower frequency • Peri-urban: shorter waits, more frequent service 3. Rationale

Public transport (matatus, shared taxis, buses) is the most common motorized transport for rural populations who don't own vehicles. The total journey time includes walking to where transport is available, waiting for a vehicle, and the in-vehicle travel - all of which make public transport substantially longer than private vehicle travel. This metric captures the realistic travel experience for most pregnant women in the study area.
